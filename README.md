# dice-rolling-game
A simple command-line dice rolling game built with Python. The game simulates rolling two dice, tracks the highest roll, counts the total number of rolls, and detects special dice combinations.

## Features

- Roll two random dice
- Display the total of both dice
- Track the total number of rolls
- Track the highest roll in the session
- Detect doubles
- Detect Snake Eyes (1,1)
- Detect Lucky 7 (sum = 7)
- Detect Boxcars (6,6)
- Handle invalid user input

## Technologies Used

- Python 3
- random module

## Requirements

- Python 3.x
